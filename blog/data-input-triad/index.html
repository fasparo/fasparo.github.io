The new triad of data input

2012-11-01

On the desktop, a physical keyboard with nubbins for light-speed touch typists. On the tablet, a software keyboard with intensive error correction to allow two-fingered input (current contender: Swiftkey). On the pocket device (also still known as smartphone), a software keyboard allowing one-handed word-swiping (current contender: Swype).
